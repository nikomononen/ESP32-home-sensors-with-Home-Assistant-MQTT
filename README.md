# ESP32-home-sensors-with-MQTT-Home-Assistant
ESP32 with sensors to sense the state of the room. Homeassistant integration with MQTT

Sensors
- AM2302
- BME280
- TSL2561
- GY-SGP30
- GUVA-S12SD
